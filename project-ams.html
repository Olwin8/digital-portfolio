Auto Machinery Singapore (AMS) E-Commerce Platform

Digitising motorbike parts sales with OCR and inventory management

Role

Full Stack Developer

Technologies

React.js, Node.js, Express.js, MySQL, OCR

Duration

5 months

Client

Auto Machinery Singapore

Project Overview

Auto Machinery Singapore (AMS) is a motorbike parts reseller specialising in Yamaha and Honda parts. They faced challenges with their PDF catalog system, making it difficult for staff to track inventory and assist customers effectively.

I developed a comprehensive solution that included OCR technology to extract parts details from PDFs, built a responsive e-commerce platform, and implemented inventory management features to streamline their sales process.

Key Features

PDF OCR Processing

Implemented OCR technology to extract parts details from thousands of PDF catalog pages, converting them into structured database entries.

E-Commerce Platform

Built a user-friendly interface allowing customers to browse and purchase parts by bike model and part category.

Mobile-First Design

Created a fully responsive design optimised for both desktop and mobile use, as staff often use mobile devices in the shop.

Inventory Management

Developed real-time inventory tracking with low-stock alerts and sales analytics dashboard.

Challenges & Solutions

PDF Data Extraction

The existing PDF catalogs had inconsistent formatting, making data extraction difficult.

Solution: Developed a custom OCR pipeline with validation checks and manual review tools to ensure data accuracy.

Legacy System Integration

AMS had existing inventory systems that needed to be integrated.

Solution: Created API endpoints to sync data between the new platform and their legacy systems.

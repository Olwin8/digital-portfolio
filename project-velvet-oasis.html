The Velvet Oasis Hotel Prototype

An interactive hotel website prototype designed with Figma, focusing on user experience and visual appeal

Role

UI/UX Designer

Tools

Figma, Prototyping, Wireframing

Duration

2 months

Category

Web Design, Prototype

Project Overview

Designed an interactive website prototype for The Velvet Oasis Hotel, a luxury destination for couples and families. The project included a home page, room & facilities page, and a booking/enquiry form, focusing on clean visuals, intuitive navigation, and essential interactive elements.

Key Pages & Features

Home/Landing Page

Showcased the hotel's luxury offerings with hero imagery, special offers, and clear call-to-action buttons to explore rooms or book directly.

Rooms & Facilities

Detailed presentation of Premier, Superior, and Royal Suite rooms with interactive image galleries and amenity highlights.

Booking Form

Streamlined reservation process with intuitive form fields, date pickers, and room selection dropdowns.

Interactive Elements

Implemented hover effects, dropdown menus, and clickable components to enhance user engagement.

Visual Design

Created a sophisticated color scheme and typography hierarchy that reflects the hotel's luxury brand.

User Flows

Designed intuitive navigation paths from discovery to booking, ensuring a seamless experience.
